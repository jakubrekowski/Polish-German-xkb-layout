# Polish (German) xkb layout

xkb Polish layout with German characters.

## instalation

1. Run as sudo ```./install.sh```,
2. Add polish layout with variant ```Polish (German)``` in keyboard settsing.

## using
```
Alt + A => Ä
Alt + a => ä
Alt + Ö => Ö
Alt + ö => ö
Alt + U => Ü
Alt + u => ü
Alt + S => ẞ
Alt + s => ß
```
