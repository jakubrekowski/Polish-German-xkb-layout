# Polski (Niemiecki) układ klawiatury xkb

Polski układ klawiatury xkb z znakami Niemieckimi.

## instalacja
1. Uruchom skrypt ```sudo ./install.sh```
2. Dodaj polski układ klawiatury w wariancie ```Polish (German)```

## używanie
```
Alt + Shift + A => Ä
Alt + a => ä
Alt + Shift + Ö => Ö
Alt + ö => ö
Alt + Shift + U => Ü
Alt + u => ü
Alt + Shift + S => ẞ
Alt + s => ß
```
